#!/usr/bin/env dotnet
#:sdk Microsoft.NET.Sdk.Web
#:property PublishAot=false

var builder = WebApplication.CreateBuilder(args);

var port = Environment.GetEnvironmentVariable("PORT") ?? "3000";
var url = $"http://0.0.0.0:{port}";
var target = Environment.GetEnvironmentVariable("TARGET") ?? "World";

using var app = builder.Build();
app.MapGet("/", () => $"Hello {target}!");
app.Run(url);

How It Works

Understanding .NET 10's file-based application magic

#!/usr/bin/env dotnet

Shebang Directive

Enables direct execution of the C# file as a script on Unix-like systems.

#:sdk Microsoft.NET.Sdk.Web

SDK Directive

Specifies the SDK to use, enabling web application features without a project file.

#:property PublishAot=false

Property Directive

Sets MSBuild properties inline. Configure your build without leaving the code file.
