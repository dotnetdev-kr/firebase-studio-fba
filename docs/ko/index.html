Program.cs 
#!/usr/bin/env dotnet
#:sdk Microsoft.NET.Sdk.Web
#:property PublishAot=false

var builder = WebApplication.CreateBuilder(args);

var port = Environment.GetEnvironmentVariable("PORT") ?? "3000";
var url = $"http://0.0.0.0:{port}";
var target = Environment.GetEnvironmentVariable("TARGET") ?? "World";

using var app = builder.Build();
app.MapGet("/", () => $"Hello {target}!");
app.Run(url);

AI 퍼스트 개발 경험

Microsoft Learn MCP 서버

Gemini AI를 공식 Microsoft 문서에 직접 연결하는 Model Context Protocol (MCP) 서버가 통합되어 있습니다. 최신 Microsoft Learn 콘텐츠를 기반으로 정확하고 최신의 .NET 10 코드 제안을 받아보세요.

  • 실시간 문서 검색
  • 항상 최신 .NET 기능 반영
  • AI 환각(hallucination) 대폭 감소

FBA 최적화 AI 프롬프트

포함된 GEMINI.md 파일에는 Gemini AI가 .NET 10 파일 기반 애플리케이션에 최적화된 코드를 생성하도록 안내하는 세심하게 작성된 프롬프트가 포함되어 있습니다.

  • .csproj 생성 없이 단일 파일 유지
  • 기본적으로 최상위 문(top-level statements) 사용
  • Minimal API 패턴 내장

왜 이 템플릿을 선택해야 할까요?

파일 기반 애플리케이션으로 .NET 개발의 미래를 경험하세요

📄

단일 파일 애플리케이션

.csproj 파일이 필요 없습니다. SDK와 MSBuild 속성은 특수 지시문을 사용하여 인라인으로 정의됩니다.

⚡

Minimal APIs

최소한의 상용구 코드로 빠른 HTTP API를 구축하세요. 마이크로서비스, 프로토타입 및 학습에 완벽합니다.

🔥

Firebase Studio 지원

Google의 Firebase Studio (IDX) 클라우드 개발 환경에서 원활하게 실행되도록 사전 구성되어 있습니다.

🚀

.NET 10

최첨단 기능과 성능 개선이 포함된 최신 .NET 10으로 구축되었습니다.

AI 기반 개발

권장 확장 설치

💡 팁: 모든 권장 확장을 설치하면 IntelliSense, 디버깅 및 AI 지원을 통한 최상의 개발 경험을 보장합니다.

4

애플리케이션 실행

▶️ 에디터 오른쪽 상단의 Run Code 버튼(▶)을 클릭하거나, Ctrl+Alt+N을 눌러 애플리케이션을 실행하세요.
5

개발 시작!

앱이 포트 3000에서 실행됩니다. 미리보기를 열고 코딩을 시작하세요!

#!/usr/bin/env dotnet

Shebang 지시문

Unix 계열 시스템에서 C# 파일을 스크립트로 직접 실행할 수 있게 합니다.

#:sdk Microsoft.NET.Sdk.Web

SDK 지시문

사용할 SDK를 지정하여 프로젝트 파일 없이 웹 애플리케이션 기능을 활성화합니다.

#:property PublishAot=false

속성 지시문

MSBuild 속성을 인라인으로 설정합니다. 코드 파일을 벗어나지 않고 빌드를 구성하세요.
